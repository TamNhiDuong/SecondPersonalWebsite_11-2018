Debugging contains many steps to go through:

  • Firstly, make sure that the files are saved.
  • Secondly, make sure that HTML file, CSS, JvS are linked.
  • Next, check if id names in HTML file and CSS, JvS files are the same.
  • Refresh the page to see the latest version.
  • Check if we are writing to the web page or to the developer console.
  • Run the Debugger in the using browser to check until which stage everything works as planned.
  • Run JavaScript code through jslint.com.
