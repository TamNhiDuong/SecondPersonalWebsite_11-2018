Topic Brief theory Difficulties
Variable
  1. JavaScript variables are used to store data values.
  2. Variables must have identifiers or unique names.
  3. Variables must have assignment operator which not have the same meaning as "equal to".
 Having misunderstanding about "Assignment operators". They do not mean "equal to" but mean assigning a value to variable.
Condition
  1. JavaScript condition is used to perform different actions based on different conditions.
  2. The "if" statement execute a code if a specific condition is true.
  3. The "else" statement execute a code if a specific condition is false.
  4. The "else if" execute a code if the first condition is false.
Using if- else statement in a loop to specify and look for an element in an array which satisfy the condition.
Function
  1. JavaScript function is a block of code designed to carry out a specific task.
  2. Function names can contain letters, digits, underscores, dollar signs.
  3. Function helps to avoid writing the same codes many times.
 Calling funcion in another function and using function in the right place.
Loop
  1. Loop is used to execute a block of code many times depending on its condition.
  2. Loop for is used to go through a block of code a numbers of time with the conditions.
  3. While loop is used to go through a block of code as long as the condition is true.
  • Loop is the most difficult part for me.
  • "How to set the condition for loop to generate the right answer" is a consideration.
  • "What will the loop print?" is also a hard part especially when I have to do the test on paper.
Array
  1. Array is used to store many values in a variable.
  2. We can access to an array element by using Index number.
  3. We can loop through an array to find elements which satisfy a specific condition.
 There are many array methods to remember. The hardest part in Array topic is combining the knowledge about Loop to go through an array.
Object
  1. Objects are variables which can contain many values.
  2. Objects define properties and value of an objective.
  3. We can access to an object by using object methods.
 Using JSON is the most difficult part of Object. We have to connect to an information source and extract information from there by using Object methods which is complicated sometimes.
Debugging
  1. Debugging is the action of searching for error, mistake in codes to fix.
  2. All Browsers nowadays have a built in debugger which support the code fixing process a lot.
  3. "Breakpoints" is where the executing of JavaScript stops for debugging.
 The code can contains both synctax and logical mistakes. But the logical mistakes is harder to be fixed as it not only requires the debugger's investigation but also programmer's understanding about their own code.
